# 图标资源说明

## 文件要求

### 1. icon.ico
- **路径**: `resources/icon.ico`
- **格式**: Windows ICO格式
- **推荐尺寸**: 包含多个尺寸的ICO文件（16x16, 32x32, 48x48, 64x64, 128x128, 256x256）
- **用途**: 
  - 应用程序主图标
  - 所有窗口的窗口图标
  - Windows任务栏图标
  - 可执行文件图标

### 2. splash_logo.png
- **路径**: `resources/splash_logo.png`
- **格式**: PNG格式（支持透明背景）
- **推荐尺寸**: 512x512 像素或更大
- **用途**: 启动画面的logo显示

## 如何创建图标

### 方法1: 使用在线工具（推荐）

1. **准备原始图像**
   - 设计一个512x512像素的图标设计
   - 保存为PNG格式

2. **转换为ICO格式**
   - 访问 https://convertio.co/zh/png-ico/ 或 https://www.icoconverter.com/
   - 上传PNG文件
   - 选择多个尺寸（16x16, 32x32, 48x48, 64x64, 128x128, 256x256）
   - 下载生成的ICO文件
   - 重命名为 `icon.ico` 并放入 `resources/` 目录

3. **启动画面Logo**
   - 使用相同的PNG文件或创建一个专门的设计
   - 保存为 `splash_logo.png` 并放入 `resources/` 目录

### 方法2: 使用专业软件

- **GIMP** (免费): 可以导出ICO格式
- **Photoshop** (付费): 可以使用插件导出ICO
- **IcoFX** (Windows): 专门的图标编辑软件
- **ImageMagick** (命令行): `convert logo.png -define icon:auto-resize=256,128,64,48,32,16 icon.ico`

## 设计建议

1. **图标设计**
   - 使用简洁、易识别的设计
   - 确保在小尺寸（16x16）时仍然清晰可辨
   - 使用与软件主题相关的颜色和元素
   - 考虑使用光谱、图表、科学仪器等相关元素

2. **颜色方案**
   - 使用高对比度颜色
   - 避免过于复杂的渐变
   - 确保在不同背景上都能清晰显示

3. **启动画面Logo**
   - 可以使用更大的设计空间
   - 可以包含软件名称和图标
   - 考虑使用渐变色和现代设计风格

## 文件放置

将文件放置在以下位置：
```
项目根目录/
└── resources/
    ├── icon.ico          # 应用程序图标
    └── splash_logo.png   # 启动画面Logo
```

## 验证

放置图标文件后，运行程序：
1. 检查窗口标题栏是否显示图标
2. 检查任务栏是否显示图标
3. 检查启动画面是否显示logo
4. 如果图标没有显示，检查控制台是否有错误消息

## 注意事项

- 如果图标文件不存在，程序仍会正常运行，只是不会显示图标
- ICO文件必须包含多个尺寸才能在不同场景正确显示
- PNG文件支持透明背景，可以让启动画面更美观
